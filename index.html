Schritt-für-Schritt Anleitung

  1. Präsenz herstellen: Erlauben Sie den Zugriff auf Ihre Standortdaten. Sie werden dann automatisch auf der interaktiven Karte angezeigt – Ihr Name wird zufällig generiert und kann bei Bedarf angepasst werden.
  2. Dateiübertragung: Wählen Sie eine Datei aus und senden Sie sie. Ihre Datei wird vor der Übertragung verschlüsselt.
  3. Chat: Schreiben Sie Nachrichten in das Chat-Feld und kommunizieren Sie in Echtzeit.
  4. Snapshot: Nehmen Sie einen Schnappschuss Ihres Webcam-Streams auf und senden Sie ihn.
